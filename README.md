# snake-game
A version of Snake written in C using SDL2 for graphics.

![snake](https://github.com/user-attachments/assets/624ead3d-44ef-49ec-8ea0-84eb44c0d84f)


# Building
This requires SDL2, instructions on how to install it can be found here:

https://wiki.libsdl.org/SDL2/Installation


After that, just go to the game's directory and type ``make`` in your terminal. I don't know if this will work on Windows or MacOS.
