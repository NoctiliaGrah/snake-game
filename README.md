# snake-game
A very unfinished, very bodged together version of Snake written in C using SDL2 for graphics.

I'm unlikely to keep working on this after I've gotten all the main Snake features implemented. Most of this code is "respectfully borrowed" from YouTube tutorials, other GitHub projects, Reddit, and whatever else makes getting it working easier on my poor little brain. I'm not a programmer. I'm not even close to one, I'm a musician that does stuff like this as a hobby, so expect nothing and I will expect nothing from you. If you're using this as a reference for your own projects, no you aren't. Use something else, something made by someone that knows what they're doing.

# Building
This requires SDL 2, instructions on how to install it can be found here:

https://wiki.libsdl.org/SDL2/Installation


After that, just type ``make`` in your terminal. If you're on Windows, good luck to you I guess.
